# Task 7. 설정 창/트레이/로컬 보안 저장소 구현

## 목표
사용자가 API 키와 동작 옵션을 변경하고, 앱을 트레이 중심으로 운용할 수 있도록 관리 기능을 구현한다.

## 범위
- 시스템 트레이 아이콘 및 메뉴:
  - 설정 열기
  - 앱 숨기기/표시
  - 종료
- 설정 창 UI:
  - Papago Client ID/Secret 입력
  - 기본 번역 언어
  - 이동 속도
  - 알림음 ON/OFF
  - 단축키 변경
- API 키 저장:
  - 우선 `keytar` 사용
  - 불가 시 암호화 로컬 저장 fallback 정의
- 설정 로드/검증/저장/적용 플로우 구현.

## 산출물
- 트레이에서 설정 접근 및 앱 제어 가능.
- 재시작 후에도 설정이 유지됨.

## 완료 기준 (DoD)
- 민감정보가 평문으로 노출되지 않는다.
- 잘못된 API 키 입력 시 명확한 오류 피드백 제공.
- 단축키 변경 후 즉시 재등록된다.
