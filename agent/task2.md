# Task 2. 투명 창/항상 위/클릭 스루 윈도우 제어 구현

## 목표
캐릭터가 데스크탑 최상단에 상주하고, 캐릭터 외 영역은 클릭이 아래 창으로 통과되는 핵심 윈도우 동작을 구현한다.

## 범위
- `BrowserWindow`에 투명 배경 및 프레임리스 설정 적용.
- 항상 위(`alwaysOnTop`) 고정.
- 캐릭터 픽셀 영역 외 클릭 스루 처리:
  - 기본은 `setIgnoreMouseEvents(true, { forward: true })`
  - 캐릭터 히트 영역 진입 시 이벤트 수신 전환.
- 멀티 모니터/화면 경계 기준 좌표 처리 준비.

## 산출물
- 캐릭터 영역만 클릭 반응, 나머지 영역은 하위 앱 클릭 가능.
- 창이 다른 앱에 가려지지 않음.

## 완료 기준 (DoD)
- 브라우저/에디터 위에서도 캐릭터가 항상 보인다.
- 캐릭터 밖 클릭 시 뒤 창이 정상 클릭된다.
- 캐릭터 위 클릭 시 이벤트가 정상 수신된다.
