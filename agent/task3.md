# Task 3. 캐릭터 상태 머신 및 이동/애니메이션 시스템

## 목표
PRD의 캐릭터 상태(대기/걷기/뛰기/번역중/말하기/놀기)를 상태 머신으로 구현하고, 화면 내 자율 이동과 애니메이션 전환을 완성한다.

## 범위
- 상태 enum 및 전이 규칙 정의.
- 화면 경계 충돌 처리 및 반사/회피 이동 로직.
- CSS 애니메이션 기반 모션 전환(`transform`, `keyframes` 중심).
- 무입력 시간 기준 자율 행동(랜덤 이동, 짧은 멘트 트리거).
- 자원 구조 정리: 상태별 스프라이트/애니메이션 매핑.

## 산출물
- 캐릭터가 자연스럽게 상태 전환하며 이동.
- Idle 시간에 자율 상호작용 발생.

## 완료 기준 (DoD)
- 각 상태 진입/이탈이 로그 또는 디버그 패널로 확인 가능.
- 경계 밖으로 캐릭터가 벗어나지 않는다.
- JS 타이머 남용 없이 CSS 애니메이션 중심으로 동작한다.
