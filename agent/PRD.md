# 제품 요구사항 정의서 (PRD)

**프로젝트명:** 데스크탑 AI 번역 펫 (가칭: TranslateMate)
**작성일:** 2026년 5월 15일
**버전:** 1.0.0

---

## 1. 개요 (Overview)

### 1.1 제품 목적
본 프로그램은 사용자의 PC 화면에 상주하며 상호작용하는 귀여운 가상 캐릭터(AI 에이전트)입니다. 사용자가 입력한 텍스트를 네이버 파파고 API를 통해 실시간으로 번역하여 캐릭터가 말풍선으로 알려주는 기능을 핵심으로 하며, 데스크탑 환경에서 작업 효율성과 시각적 즐거움을 동시에 제공하는 것을 목표로 합니다.

### 1.2 타겟 사용자
* PC 작업 중 웹 브라우저를 켜지 않고 즉각적인 텍스트 번역이 자주 필요한 사용자
* 논문, 해외 자료 등을 읽으면서 실시간 보조가 필요한 학생 및 연구자
* 단조로운 데스크탑 환경에 소소한 재미와 캐릭터의 상호작용을 원하는 사용자

---

## 2. 핵심 기능 명세 (Key Features)

### 2.1 캐릭터 시스템 및 애니메이션
* **상태 기반 애니메이션:** 캐릭터는 '대기', '걷기', '뛰기', '번역 중(생각하기)', '말하기', '놀기' 등의 상태를 가집니다.
* **자유로운 화면 이동:** 캐릭터는 화면의 경계를 인식하며, 바탕화면이나 다른 창들의 위를 자유롭게 배회합니다.
* **자율 상호작용:** 일정 시간 사용자의 입력이 없으면 캐릭터가 무작위로 혼자 놀거나 짧은 인사말, 격려의 말(미리 정의된 텍스트)을 말풍선으로 띄웁니다.
* **마우스 상호작용:** 마우스 클릭 시 캐릭터가 반응(깜짝 놀라기, 인사하기 등)하며, 드래그 앤 드롭으로 화면 내 위치를 강제로 이동시킬 수 있어야 합니다.

### 2.2 파파고 API 연동 번역 기능
* **입력 인터페이스:** 
  * 캐릭터 우클릭 메뉴 또는 지정된 단축키(예: `Ctrl + Shift + T`)를 통해 텍스트 입력창(팝업)을 호출합니다.
  * 복사된 텍스트가 있는 상태에서 단축키를 누르면 자동으로 클립보드 내용을 가져와 번역하는 기능을 지원합니다.
* **번역 처리:** 
  * 사용자가 문장을 입력하면 네이버 클라우드 Papago NMT API를 호출하여 번역을 수행합니다.
  * 출발어(자동 감지) 및 도착어(기본 설정 언어, 예: 한국어/영어)를 지원합니다.
* **결과 출력:** 번역이 완료되면 캐릭터의 말풍선 UI를 통해 번역된 텍스트가 출력됩니다. 텍스트가 길 경우 타이핑 효과(글자가 하나씩 나타나는 효과)와 함께 스크롤 기능을 제공합니다.

### 2.3 윈도우 시스템 제어 (Always On Top)
* **항상 위 (Topmost):** 캐릭터 창은 웹 브라우저, 문서 편집기 등 다른 모든 애플리케이션 창보다 항상 최상단에 렌더링되어 가려지지 않아야 합니다.
* **투명 배경 (Transparent Window):** 캐릭터의 스프라이트(이미지) 외의 영역은 완전히 투명하게 처리되어, 마우스 클릭이 캐릭터를 관통하여 뒤에 있는 일반 창에 닿을 수 있어야 합니다(Click-through). 캐릭터 영역에만 마우스 이벤트가 발생해야 합니다.

## 3. 사용자 인터페이스 (UI/UX)
* **캐릭터 스프라이트:** 2D 픽셀 아트 또는 부드러운 2D 일러스트 기반.
* **말풍선 (Speech Bubble):** 캐릭터 머리 위나 옆에 동적으로 크기가 변하는 말풍선 생성. 가독성 높은 폰트 사용.
* **입력 프롬프트:** 미니멀하고 직관적인 텍스트 입력 UI.
* **설정 (Settings) 창:** 시스템 트레이 아이콘을 통해 접근 가능.
  * 파파고 API 키(Client ID, Client Secret) 입력란
  * 기본 번역 언어 설정
  * 캐릭터 이동 속도 및 알림음 켜기/끄기 설정
  * 단축키 변경

## 4. 기술 요구사항 (Technical Requirements)

### 4.1 권장 기술 스택 (크로스 플랫폼 지원 고려 시)
* **프레임워크:** Electron.js (웹 기술 기반, 투명 창 및 시스템 트레이 제어 용이) 또는 Tauri (Rust 기반, 가볍고 성능이 뛰어남).
* **언어/UI:** HTML, CSS, JavaScript (React 또는 Vanilla JS).
* **대안 (Python 선호 시):** PyQt5 또는 Tkinter (투명 배경 및 최상단 고정 기능 구현 가능).

### 4.2 외부 API 연동
* **API:** Naver Cloud Platform - Papago Text Translation API
* **네트워크 통신:** REST API(POST 요청)를 통한 JSON 데이터 송수신.

### 4.3 보안 및 데이터 처리
* 사용자의 API 키는 로컬 저장소(암호화된 파일 또는 시스템 Keytar)에 안전하게 저장되어야 합니다.
* 번역 기록은 로컬에만 임시 저장되며, 외부 서버로 수집되지 않아야 합니다.

## 5. 향후 확장 및 제안 기능 (Future Scope)
사용자님의 주 사용 목적(논문 분석 등)을 고려하여 다음과 같은 기능을 추가해 볼 수 있습니다.

* **아카데믹 모드 (논문 분석 보조):** 
  * 파파고 번역 외에 LLM(Gemini API 등)을 추가 연동합니다.
  * 영문 논문의 복잡한 문장을 클립보드에 복사하고 호출하면, 캐릭터가 1) 문장 구조 분석, 2) 직역/의역, 3) 전문 용어 정리를 말풍선 또는 별도의 요약 창으로 깔끔하게 정리해 줍니다.
* **TTS (Text-to-Speech) 연동:** 번역된 문장이나 해석된 결과를 캐릭터가 실제 음성으로 읽어주는 기능.
* **커스텀 스킨 시스템:** 사용자가 직접 그린 이미지로 캐릭터를 교체할 수 있는 기능.
