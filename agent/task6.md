# Task 6. 말풍선 출력 시스템 및 상호작용 UX 구현

## 목표
번역 결과/자율 멘트를 캐릭터 말풍선으로 자연스럽게 출력하고, 사용자 상호작용(클릭/드래그) 피드백을 완성한다.

## 범위
- 말풍선 컴포넌트 구현 (동적 크기, 가독성 폰트, 위치 추적).
- 긴 텍스트 처리:
  - 타이핑 효과
  - 내부 스크롤
- 캐릭터 클릭 반응(인사, 놀람 등) 상태 트리거.
- 드래그 앤 드롭 위치 이동 및 드롭 후 상태 복귀.
- 번역 중/완료 상태에 맞춘 시각 피드백.

## 산출물
- 번역 결과가 말풍선에 안정적으로 출력.
- 캐릭터 클릭/드래그 상호작용이 체감 가능한 수준으로 동작.

## 완료 기준 (DoD)
- 짧은/긴 문장 모두 레이아웃 깨짐 없이 표시된다.
- 드래그 중 프레임 드랍 없이 이동된다.
- 클릭 반응이 1회 입력당 중복 트리거되지 않는다.
