# Task 4. 번역 입력 UX (단축키/우클릭/클립보드) 구현

## 목표
사용자가 빠르게 번역을 호출할 수 있도록 입력 진입점을 통합한다.

## 범위
- 글로벌 단축키 기본값 등록: `Ctrl+Shift+T` (OS별 조합 대응 포함).
- 캐릭터 우클릭 컨텍스트 메뉴에서 "번역 입력" 제공.
- 입력 팝업 UI(미니멀 텍스트 입력창) 구현.
- 단축키 호출 시 클립보드 텍스트 자동 로드 옵션 지원.
- 입력 검증(빈 문자열/길이 제한/공백 처리).

## 산출물
- 단축키 또는 우클릭으로 즉시 입력창 호출 가능.
- 클립보드 기반 번역 시작 가능.

## 완료 기준 (DoD)
- 단축키 충돌/등록 실패 시 사용자에게 안내된다.
- 입력창에서 Enter로 번역 요청, Esc로 취소 가능.
- 클립보드 비어있을 때 예외 없이 동작한다.
